# CS Internship Portfolio (React + Vite)

1. Install dependencies:

```bash
npm install
```

2. Run the dev server:

```bash
npm run dev
```

- `package.json` - basic scripts and deps
- `index.html` - app entry
- `src/main.jsx` - React entry
- `src/App.jsx` - root component
- `src/components/*` - `Header`, `About`, `Projects`, `Contact`
- `src/styles.css` - simple responsive styling

- Replace placeholder text with your details and projects
- Add images, project links and a downloadable resume
